# Intrusion Detection System (IDS) and its Types

## Introduction

An **Intrusion Detection System (IDS)** is a security tool that monitors network traffic and system activities to detect unauthorized access, attacks, or suspicious behavior. It helps organizations identify security threats and take corrective actions before serious damage occurs.

## Definition

An IDS is a hardware or software system that continuously monitors a computer system or network and generates alerts when it detects malicious activities or policy violations.

## Functions of IDS

* Monitor network and system activities.
* Detect unauthorized access attempts.
* Identify malware, viruses, and attacks.
* Generate alerts and reports.
* Help administrators respond to security incidents.

## Types of Intrusion Detection System (IDS)

### 1. Host-Based Intrusion Detection System (HIDS)

* Installed on an individual computer or server.
* Monitors system files, logs, processes, and user activities.
* Detects unauthorized changes within the host system.

**Example:** Monitoring changes in critical operating system files.

**Advantages:**

* Detects internal attacks.
* Provides detailed information about the affected system.

### 2. Network-Based Intrusion Detection System (NIDS)

* Installed at strategic points within a network.
* Monitors incoming and outgoing network traffic.
* Detects suspicious network activities and attacks.

**Example:** Detecting port scanning or Denial of Service (DoS) attacks.

**Advantages:**

* Protects multiple devices simultaneously.
* Monitors entire network traffic.

## IDS Based on Detection Method

### 3. Signature-Based IDS

* Detects attacks by comparing activities with known attack signatures.
* Effective against known threats.

**Advantages:**

* Fast and accurate for known attacks.

**Disadvantages:**

* Cannot detect new or unknown attacks.

### 4. Anomaly-Based IDS

* Detects unusual behavior by comparing activities with normal system behavior.
* Can identify unknown attacks.

**Advantages:**

* Detects new and zero-day attacks.

**Disadvantages:**

* May generate false alarms.

## Benefits of IDS

1. Detects security threats early.
2. Monitors network and system activities continuously.
3. Helps prevent data breaches.
4. Supports incident response.
5. Improves overall security posture.

## Conclusion

An **Intrusion Detection System (IDS)** is an essential security mechanism that detects unauthorized activities and security threats. The main types are **Host-Based IDS (HIDS)** and **Network-Based IDS (NIDS)**, while detection methods include **Signature-Based** and **Anomaly-Based** IDS. Together, they help organizations maintain a secure computing environment.
