# Cryptography and Related Terms

## Introduction

**Cryptography** is the science of protecting information by converting it into a secret form so that only authorized users can read it. It is widely used to secure data during storage and transmission over networks.

## Definition

Cryptography is the process of transforming **plaintext** (readable data) into **ciphertext** (unreadable data) using an encryption algorithm and a key.

### Basic Process

```text
Plaintext → Encryption → Ciphertext
Ciphertext → Decryption → Plaintext
```

## Related Terms in Cryptography

### 1. Plaintext

* Original readable message or data.
* Input to the encryption process.

**Example:** "HELLO"

### 2. Ciphertext

* Encrypted form of plaintext.
* Cannot be understood without a key.

**Example:** "X7P9K2"

### 3. Encryption

* Process of converting plaintext into ciphertext.
* Protects data from unauthorized access.

### 4. Decryption

* Process of converting ciphertext back into plaintext.
* Performed using a decryption key.

### 5. Cipher

* Mathematical algorithm used for encryption and decryption.

**Examples:** AES, DES, RSA.

### 6. Key

* A secret value used in encryption and decryption.
* Security of encrypted data depends on the key.

### 7. Cryptanalysis

* Study of breaking encryption and discovering hidden information without knowing the key.

### 8. Cryptology

* The overall study of cryptography and cryptanalysis.

### 9. Symmetric Key Cryptography

* Same key is used for both encryption and decryption.
* Fast and efficient.

**Example:** AES, DES.

### 10. Asymmetric Key Cryptography

* Uses two keys:

  * Public Key (for encryption)
  * Private Key (for decryption)

**Example:** RSA.

### 11. Digital Signature

* Used to verify the authenticity and integrity of a message.
* Ensures that the sender is genuine.

### 12. Hash Function

* Converts data into a fixed-length hash value.
* Used for data integrity verification.

**Examples:** SHA-256, MD5.

## Advantages of Cryptography

1. Protects confidential information.
2. Ensures data integrity.
3. Provides authentication.
4. Supports non-repudiation.
5. Secures online transactions and communications.

## Applications of Cryptography

* Online banking
* E-commerce transactions
* Secure emails
* Digital signatures
* Password protection
* Network security

## Conclusion

Cryptography is an essential security technique used to protect information from unauthorized access. Through encryption, decryption, keys, digital signatures, and hash functions, it ensures confidentiality, integrity, authentication, and secure communication in modern information systems.
