# Compare ISO 17799 and BS 7799

**Introduction:**
ISO 17799 and BS 7799 are standards related to Information Security Management. BS 7799 was developed in the UK, while ISO 17799 is its international version.

| Basis              | BS 7799                                                                     | ISO 17799                                                                   |
| ------------------ | --------------------------------------------------------------------------- | --------------------------------------------------------------------------- |
| **Full Form**      | British Standard 7799                                                       | International Standard 17799                                                |
| **Developed By**   | British Standards Institution (BSI), UK                                     | International Organization for Standardization (ISO)                        |
| **Scope**          | Provides framework for Information Security Management System (ISMS)        | Provides guidelines and best practices for information security management  |
| **Origin**         | Introduced in 1995 in the United Kingdom                                    | Published by ISO in 2000 based on BS 7799 Part 1                            |
| **Recognition**    | National standard of the UK                                                 | Internationally accepted standard                                           |
| **Purpose**        | Defines requirements and controls for security management                   | Gives recommendations for implementing security controls                    |
| **Certification**  | BS 7799 Part 2 allowed certification of ISMS                                | ISO 17799 itself was mainly a code of practice and not directly certifiable |
| **Coverage**       | Security policy, risk management, access control, business continuity, etc. | Same security domains and controls as BS 7799 Part 1                        |
| **Current Status** | Replaced by ISO 27001 and related standards                                 | Renamed and evolved into ISO/IEC 27002                                      |

### Relationship Between ISO 17799 and BS 7799

* BS 7799 was the original British standard.
* ISO adopted BS 7799 Part 1 and published it as ISO 17799.
* Later, ISO 17799 was revised and renamed ISO/IEC 27002.
* BS 7799 Part 2 became the basis for ISO/IEC 27001.

### Conclusion

BS 7799 is the original British information security standard, whereas ISO 17799 is its internationally recognized version. Both provide guidelines for managing information security, and they later evolved into the ISO 27000 family of standards widely used today.
