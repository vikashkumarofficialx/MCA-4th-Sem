# Explain NSTISSC Security Model

## Introduction

The **NSTISSC Security Model** was developed by the **National Security Telecommunications and Information Systems Security Committee (NSTISSC)**. It is also known as the **McCumber Cube Model**.

This model provides a comprehensive framework for understanding and managing information security. It shows the relationship between security objectives, information states, and security measures.

---

## NSTISSC (McCumber Cube) Model

The model is represented as a **three-dimensional cube** consisting of three components:

### 1. Security Objectives

These define what needs to be protected.

* **Confidentiality** – Information should be accessible only to authorized users.
* **Integrity** – Information should remain accurate and unaltered.
* **Availability** – Information should be available whenever required.

---

### 2. Information States

Information exists in three states:

* **Storage** – Data stored in databases, hard disks, cloud storage, etc.
* **Processing** – Data being processed by computers and applications.
* **Transmission** – Data being transferred through networks or communication channels.

---

### 3. Security Measures

These are methods used to protect information.

* **Technology** – Firewalls, encryption, antivirus software, etc.
* **Policies and Practices** – Security rules, procedures, and guidelines.
* **Education, Training, and Awareness** – Teaching users safe security practices.

## Working of the Model

The NSTISSC model states that:

* Every piece of information must be protected in all three states (storage, processing, and transmission).
* Security objectives (Confidentiality, Integrity, Availability) must be maintained for each state.
* Appropriate security measures (technology, policies, and awareness) should be applied to achieve protection.

Thus, the model helps organizations analyze security from all possible angles.

---

## Advantages of NSTISSC Security Model

1. Provides a complete view of information security.
2. Helps identify security weaknesses.
3. Covers technical and non-technical security controls.
4. Useful for designing and evaluating security programs.
5. Ensures protection of data throughout its lifecycle.

---

## Conclusion

The **NSTISSC Security Model (McCumber Cube)** is a three-dimensional information security framework that combines **Confidentiality, Integrity, Availability**, **Storage, Processing, Transmission**, and **Technology, Policies, Awareness**. It helps organizations implement effective and comprehensive information security.
